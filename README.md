# BashLab1

### Команда:
```bash
tail -n 40 file1 > file2 && head file2 > file3 && cat file2 | grep коко > tempfile && sed -i 's/коко/куку/g' tempfile && head -n 3 tempfile >> file3 && rm -f tempfile && sort file3 | uniq -c && sort -u file3 -o file3
```
